1.Class:-

Class helps us to create object when it receives request from new keyword.

2.new:-

It sends request to the class to create object.

once object is created it will get the object's address and stores that in a reference variable.

for example:

public class A{
public static void main (String[] args){
A a1=new A();
System.out.println(a1);
A a2=new A();
System.out.println(a2);
A a3=new A();
System.out.println(a3);
}
}
